✅ ③ Notifications soumission devoir

Problème : Aucune notification au formateur lors d'une soumission de devoir.

Solution : Table notifications + trigger PostgreSQL + colonnes email/WhatsApp dans personnel.

Détails 
-- Table de notifications
CREATE TABLE IF NOT EXISTS public.notifications (
  id INT GENERATED ALWAYS AS IDENTITY PRIMARY KEY,
  destinataire_id INT REFERENCES public.personnel(id) ON DELETE CASCADE,
  type TEXT NOT NULL CHECK (type IN ('whatsapp','email')),
  sujet TEXT NOT NULL,
  message TEXT NOT NULL,
  statut TEXT DEFAULT 'pending',
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Trigger automatique à la soumission
CREATE OR REPLACE FUNCTION public.notifier_soumission_devoir()
RETURNS TRIGGER AS $$
BEGIN
  INSERT INTO public.notifications (destinataire_id, type, sujet, message)
  SELECT p.id, 'email',
    'Nouveau devoir soumis',
    'Un étudiant a soumis un devoir. Connectez-vous pour le consulter.'
  FROM public.personnel p
  WHERE p.id = NEW.formateur_id AND p.notification_email = true;
  RETURN NEW;
END;
$$ LANGUAGE plpgsql SECURITY DEFINER;

✅ ⑦ Timer quiz dashboard

Problème : Absence de timer dans les quiz intégrés au dashboard.

Solution : Implémentation d'un timer avec apTimerStart, qTimerInit, barre de progression + configuration via dashboard.

Détails d'implémentation 
// Initialisation du timer
function qTimerInit(dureeSecondes, callback) {
  if (window.quizTimerInterval) clearInterval(window.quizTimerInterval);
  const startTime = Date.now();
  const endTime = startTime + (dureeSecondes * 1000);
  
  window.quizTimerInterval = setInterval(() => {
    const remaining = Math.max(0, Math.floor((endTime - Date.now()) / 1000));
    const minutes = Math.floor(remaining / 60);
    const secondes = remaining % 60;
    
    document.getElementById('timerDisplay').textContent = 
      `${minutes}:${secondes.toString().padStart(2, '0')}`;
    document.getElementById('timerBar').style.width = 
      `${(remaining / dureeSecondes) * 100}%`;
    
    if (remaining <= 0) { clearInterval(window.quizTimerInterval); callback(); }
  }, 1000);
}
