📱 Enregistrer l'écran avec le son
Votre navigateur ne supporte pas la capture d'écran web. Utilisez l'enregistrement natif de votre appareil en suivant les étapes ci-dessous.
1
Ouvrir les raccourcis rapides
Glissez deux fois vers le bas depuis le haut de l'écran pour afficher tous les raccourcis.
2
Appuyez sur "Enregistrement d'écran"
Si absent, appuyez sur ✏️ Modifier → faites glisser "Enregistrement d'écran" dans la liste active.
3
Activer "Sons du support" (SON INTERNE)
Dans la fenêtre de démarrage → sélectionnez "Sons du support" ou "Sons de l'appareil". NE PAS choisir "Microphone" seul → aucun son vidéo.
4
Revenir sur la Live Class → lancer l'enregistrement
Un compte à rebours de 3s démarre. Revenez sur cet onglet et jouez la vidéo.
⚠ YouTube peut bloquer l'audio sur certains appareils Android (protection DRM). Si le son manque, essayez la qualité vidéo 360p ou 480p dans le lecteur YouTube.
✅ Pour arrêter : appuyez sur la bulle rouge en haut de l'écran → "Arrêter". La vidéo est sauvegardée dans votre galerie.
1
Panneau de notifications → "Enreg. écran"
Glissez vers le bas → cherchez l'icône Enreg. écran. Si absente : appuyez sur ⋮ → "Modifier les boutons".
2
Paramètres son → choisir "Sons des médias"
La popup demande : Aucun son / Sons des médias / Sons des médias et micro. Sélectionnez "Sons des médias" pour capturer le son YouTube.
3
Appuyer sur "Démarrer l'enregistrement"
Compte à rebours 3s → revenez sur la Live Class et jouez la vidéo.
4
Arrêter via la barre de notification
Touchez la notification "Enreg. écran en cours" → "Arrêter". Fichier dans Galerie → Enregistrements d'écran.
⚠ Sur One UI 5+, si "Sons des médias" ne capture pas YouTube : Paramètres → Fonctions avancées → Enregistrement d'écran → activez "Son des médias".
1
Ajouter l'enregistrement au Centre de contrôle
Réglages → Centre de contrôle → appuyez sur ＋ à côté de "Enregistrement de l'écran" (une seule fois).
2
Activer le micro (pour voix + son vidéo)
Ouvrez le Centre de contrôle → maintenez appuyé ⏺ → activez "Micro" si vous voulez aussi votre voix. Le son interne (vidéo) est toujours inclus par défaut sur iOS.
3
Lancer l'enregistrement
Touchez ⏺ dans le Centre de contrôle → compte à rebours 3s → revenez sur la Live Class.
4
Arrêter
Touchez la barre rouge en haut → "Arrêter". Ou Centre de contrôle → ⏺ à nouveau. Vidéo dans Photos → Récents.
⚠ Sur iOS, le son interne des vidéos DRM (Netflix, etc.) est bloqué. YouTube via navigateur est généralement capturé correctement.
✅ Sur iPad, le Centre de contrôle s'ouvre depuis le coin supérieur droit de l'écran.
