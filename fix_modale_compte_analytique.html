🖥️ 1. Correction de la Modale Compte Analytique

La modale addCompteModal ou addCompteAnalytiqueModal doit être corrigée pour :

  • Structure complète : header avec titre, body avec formulaire, footer avec boutons
  • Champs requis : code, libellé, type (charge/produit), description, actif
  • Validation : code obligatoire, libellé obligatoire
  • Z-index > 99999 pour être au-dessus de tout overlay
  • Animation : fade-in avec backdrop flouté
  • Accessibilité : aria-labels, focus trap, role="dialog"
