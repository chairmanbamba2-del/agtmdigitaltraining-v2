Notre Mission

AGTM Digital Academy est une plateforme numérique de nouvelle génération dédiée à l'enseignement de l'anglais en Afrique de l'Ouest. Section 05
Fonctionnalités Technologiques Clés
Les innovations distinctives qui différencient AGTM Digital Academy
🎓
Classe Virtuelle Synchronisée — Technologie Propriétaire
Synchronisation en temps réel de lecteurs YouTube entre l'enseignant et les apprenants via Supabase Realtime Broadcast. Le professeur contrôle la vidéo depuis son appareil, chaque apprenant suit exactement au même instant — sans partage d'écran, sans dépendance à Google Meet ou Zoom. L'audio est lu localement sur chaque appareil pour une qualité optimale. Sessions isolées par code unique.
❓
Quiz Triggers Automatiques — Évaluation Inline
Pendant la lecture d'une vidéo, lorsque le lecteur atteint un moment clé prédéfini (trigger_time), la lecture se suspend automatiquement et un quiz plein écran s'affiche. L'apprenant répond (MCQ ou question ouverte), reçoit un feedback immédiat et reprend la vidéo. Les quiz sont liés aux vidéos par leur ID YouTube et configurables par niveau CEFR.
🤖
Assistant IA Pédagogique — AGTM AI (Claude Anthropic)
Chaque apprenant dispose d'une assistante IA personnalisée, disponible 24h/24, propulsée par les modèles Claude (Anthropic). Elle répond aux questions d'anglais, propose des exercices adaptés, corrige des textes, explique des règles grammaticales et peut effectuer des recherches web en temps réel. 3 niveaux de service : Essentiel (Haiku), Recommandé (Sonnet), Premium (Opus).
🔴
Screen Recording & Web Share — Capture de cours
Les apprenants peuvent enregistrer des extraits de cours directement depuis le navigateur (API MediaRecorder + getDisplayMedia), incluant l'audio système (son YouTube inclus). Un point rouge clignotant signale l'enregistrement actif. Le fichier .webm se télécharge automatiquement. Le bouton "Partager" utilise le Web Share API natif pour envoyer le lien de session via WhatsApp, Email ou SMS en un tap.
📄
Génération PDF Intelligente — Rapports & Honoraires Synchronisés
L'admin, la direction et les formateurs génèrent des rapports PDF complets directement depuis le tableau de bord. Chaque rapport inclut le contenu pédagogique ET les honoraires calculés automatiquement (présents × tarif, absents NJ × 50% tarif). La vue v_rapport_honoraires_sync (base de données) garantit une cohérence temps réel entre les deux.
📊
Suivi Financier Temps Réel — Finance & Honoraires
Tableau de bord financier complet : encaissements, dépenses, caisse, échéanciers, proforma, finances personnelles de la direction. Honoraires des formateurs calculés automatiquement à chaque rapport soumis — avec statut En attente → Validé → Payé. Toutes les données sont isolées par rôle via les politiques RLS PostgreSQL.
📲
Intégration WhatsApp Business
Envoi de messages automatiques via l'API WhatsApp Cloud pour les confirmations d'inscription, rappels de séance, notifications de paiement et communication marketing. Webhook entrant pour centraliser les demandes des prospects directement dans le tableau de bord admin.
📱
Progressive Web App (PWA) — Installable sans App Store
La plateforme est installable sur Android, iOS et PC depuis le navigateur, sans passer par Google Play ou l'App Store. Mode hors-ligne pour les pages visitées, mise à jour automatique, notifications push (infrastructure en place), icône sur l'écran d'accueil. Expérience native sans développement mobile natif.
🔐
Sécurité Multi-couche — RLS & JWT
Chaque utilisateur ne voit que ses propres données : les politiques Row Level Security (RLS) de PostgreSQL sont appliquées au niveau de la base de données — pas seulement dans le code applicatif. Authentification par token JWT (Supabase Auth), clés API protégées côté serveur (proxy Netlify Functions), HTTPS obligatoire.

Section 09
Sécurité & Conformité
Protection des données à tous les niveaux de la plateforme
🔑
Authentification JWT
Chaque session utilisateur est protégée par un token JSON Web Token signé côté serveur (Supabase Auth). Expiration automatique, révocation possible à tout moment par l'admin.
🛡️
Row Level Security (RLS)
Sécurité au niveau base de données PostgreSQL. Un étudiant ne peut jamais lire les données d'un autre — même si le code applicatif avait un bug. Politique définie par rôle pour chaque table.
🔒
HTTPS & Clés Protégées
Toutes les communications sont chiffrées HTTPS (fourni par Netlify). Les clés API (Anthropic, WhatsApp) ne sont jamais exposées au navigateur — elles transitent uniquement via les fonctions serverless.
👥
Système de Rôles RBAC
6 rôles avec permissions granulaires. L'administrateur peut créer des rôles personnalisés et en assigner à n'importe quel utilisateur. Modification des accès en temps réel sans redémarrage.
🌐
CDN Mondial
La plateforme est servie depuis les points de présence Netlify les plus proches de chaque utilisateur. Disponibilité >99.9%, protection DDoS intégrée.
📋
Audit & Traçabilité
Toutes les actions critiques (inscriptions, paiements, modifications de notes) sont horodatées et attribuées à l'utilisateur qui les a effectuées. Historique complet consultable par l'admin.
