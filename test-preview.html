🚀 Actions de test recommandées

  1. Ouvrir la nouvelle vitrine dans un navigateur
  2. Tester la navigation (clic sur les liens du menu)
  3. Vérifier le responsive (redimensionner la fenêtre)
  4. Tester les liens vers le dashboard

  6. Tester le menu mobile (fenêtre < 768px)
